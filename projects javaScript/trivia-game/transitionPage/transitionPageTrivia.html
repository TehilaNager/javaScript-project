משחק טריוויה 🎯🧠

משחק טריוויה אינטראקטיבי וחווייתי שבו המשתמש בוחר קטגוריה, רמת קושי וזמן לכל שאלה – ויוצא לענות על 10 שאלות אמריקאיות ברצף. מטרת המשחק היא לצבור כמה שיותר נקודות על ידי בחירה בתשובות הנכונות.

המשחק כולל עיצוב רספונסיבי, טיימר מובנה, אפקטים קוליים ופופ־אפים עם הוראות ותוצאה סופית – כל אלה יוצרים חוויית למידה אינטואיטיבית ומהנה. למרות שהמשחק באנגלית, הוא ברור וקל להבנה גם לדוברי עברית, בזכות ההנחיות החזותיות והקוליות והחיווי המיידי לתשובות נכונות או שגויות - לחוויית למידה מהנה ואינטואיטיבית.

המשחק נבנה באמצעות הטכנולוגיות הבאות:

  • ✔️ HTML ו-CSS – לבניית מבנה המשחק ועיצוב רספונסיבי
  • ✔️ JavaScript – לניהול הלוגיקה, הטיימר, הניקוד וחוויית המשתמש
  • ✔️ Bootstrap – לעיצוב אלמנטים ולפריסת מבנה רספונסיבית
  • ✔️ SweetAlert2 – להצגת חלונות קופצים ייחודיים
  • ✔️ Open Trivia DB API – לשליפת שאלות בזמן אמת לפי בחירת המשתמש
