אפליקציית מזג האוויר ☀️🌧️

אפליקציה אינטראקטיבית לבדיקת מזג האוויר בכל עיר בעולם. המשתמש מקליד את שם העיר הרצויה, והאפליקציה מציגה את הטמפרטורה הנוכחית, תיאור מזג האוויר ואייקון ייצוגי – הכל בזמן אמת ובממשק ברור ונעים.

האפליקציה באנגלית, אך ניתן להזין בה גם שמות של ערים בעברית וגם באנגלית. היא כוללת טיפול בשגיאות, פידבק חזותי מיידי, ועיצוב רספונסיבי שמותאם למשתמשים בכל גיל. ניתן לבצע חיפוש גם בלחיצת כפתור וגם דרך מקש Enter – לנוחות מרבית.

האפליקציה נבנתה באמצעות הטכנולוגיות הבאות:

  • ✔️ HTML ו-CSS – לבניית מבנה ועיצוב האפליקציה
  • ✔️ JavaScript – לטיפול בקלט מהמשתמש, שליפת מידע ועדכון הדף
  • ✔️ OpenWeatherMap API – להצגת נתוני מזג אוויר בזמן אמת
  • ✔️ SweetAlert2 – להצגת הודעות שגיאה והודעות מידע בצורה גרפית ונוחה
