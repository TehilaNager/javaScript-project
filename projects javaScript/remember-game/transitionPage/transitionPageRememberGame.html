משחק זיכרון 🧠🐾

משחק זיכרון מהנה ומעוצב שבו על המשתמש להתאים זוגות של תמונות של חיות מתוך לוח משחק צבעוני ודינמי. בכל התחלה של משחק נשלפות תמונות חדשות באופן אקראי מ-Unsplash, כך שכל משחק הוא חוויה חדשה ובלתי צפויה.

המשחק כולל אנימציות חלקות של כרטיסים מתהפכים, אפקטים קוליים לתשובות נכונות ושגויות, וחלונות קופצים שמעודדים את השחקן להמשיך לשחק. הממשק כולו בעברית, והוא מותאם לחוויית משתמש נעימה וידידותית.

המשחק נבנה באמצעות הטכנולוגיות הבאות:

  • ✔️ HTML ו-CSS – לבניית מבנה המשחק ועיצוב הכרטיסים
  • ✔️ JavaScript – לניהול הלוגיקה של המשחק, טיפול בהתאמות ובשליפת התמונות
  • ✔️ SweetAlert2 – להצגת הודעות קופצות בסיום משחק
  • ✔️ API של Unsplash – לשליפת תמונות חיות בזמן אמת
